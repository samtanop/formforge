# FormForge PT

PT session management app for Coach Sam.

## Deploy to Vercel (step by step)

### 1. Upload to GitHub
1. Go to github.com and sign in (or create a free account)
2. Click the **+** icon top right → **New repository**
3. Name it `formforge`, set to Private, click **Create repository**
4. Click **uploading an existing file**
5. Drag and drop the entire contents of this folder (all files and the src/ folder)
6. Click **Commit changes**

### 2. Deploy on Vercel
1. Go to vercel.com and click **Sign Up with GitHub**
2. Click **Add New Project**
3. Find and select your `formforge` repository
4. Leave all settings as default — Vercel auto-detects Vite/React
5. Click **Deploy**
6. In ~60 seconds you'll get a live URL like `formforge.vercel.app`

### Updating the app
Whenever you get a new version of App.jsx from Claude:
1. Go to your GitHub repository
2. Click into `src/` → click `App.jsx` → click the pencil (edit) icon
3. Select all, paste the new code
4. Click **Commit changes**
5. Vercel redeploys automatically within 30 seconds
